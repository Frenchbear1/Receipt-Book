# Receipt Book

A lightweight, single‑file web app for logging job receipts with quick presets, paid grouping, and instant local caching.

## Features
- Email/password sign‑in (Firebase Auth)
- Job presets with fixed amounts
- One‑time quick logs using the main inputs
- Tap a job to log date/time and amount
- Log list grouped into unpaid and paid batches
- Mark current batch as paid
- Long‑press paid group header to unpay or delete the group
- Long‑press a log to delete it
- Editable log date/time via tap
- Instant local caching for fast reloads
- Floating total of all paid logs
- Mobile‑friendly layout

## Files
- `index.html` — the entire app (HTML/CSS/JS)
